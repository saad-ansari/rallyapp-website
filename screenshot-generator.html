How to Export Screenshots

  1. Click on any screenshot preview to open it at full size
  2. Use your browser's screenshot tool or Cmd+Shift+4 (Mac) / Win+Shift+S (Windows)
  3. Or right-click → "Take Screenshot" in Firefox, or use Chrome DevTools

Export Tips

For pixel-perfect exports at exact dimensions, use these methods:

Option 1: Browser DevTools

  1. Open Chrome DevTools (F12)
  2. Click device toolbar icon (Ctrl+Shift+M)
  3. Set custom dimensions (1290x2796 for iOS)
  4. Right-click → Capture screenshot

Option 2: Puppeteer Script

npx puppeteer screenshot --url file://path/to/screenshot-generator.html --viewport 1290x2796 --output screenshot.png
