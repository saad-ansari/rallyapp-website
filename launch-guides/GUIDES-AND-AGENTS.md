### Use Launch Guides When...
- You want **actionable checklists** ("Do this, then this")
- You need **templates** (app descriptions, tweets, emails)

**1. Guides first, agents second**
- Execute with guides â†’ Validate with agents
- Don't ask agents "How do I create a Twitter account?" (guides cover this)
- DO ask agents "What should I post on Twitter?" (strategic decision)

**2. Agents complement each other**
- Mentor â†’ Routes you to right specialist
- Specialists â†’ Deep dive on their domain
- Example: "Mentor: 'For TikTok strategy, consult Content Specialist'" â†’ "Content Specialist: 'Here are 20 video ideas'"

**3. Use guides as checklists during agent conversations**
- Agent gives strategy â†’ You execute using guide's checklist
- Example: "Agent: 'Do keyword research' â†’ You: Follow [Guide 04, Step 1] for exact steps"

**4. Agents adapt, guides are fixed**
- Guides = proven process (doesn't change)
- Agents = custom advice for your situation
- Example: Guide says "Launch Wednesday", Agent says "Delay 1 week because of beta bugs"
